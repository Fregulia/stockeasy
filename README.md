# StockEasy

Este projeto é um sistema de gerenciamento de produtos que permite o cadastro e exclusão de produtos, bem como a geração de relatórios detalhados sobre movimentações, estoque e produtos cadastrados.

---

## Funcionalidades

### Gestão de Produtos
- **Adicionar produtos** com nome, categoria, quantidade, preço e outras informações.
- **Excluir produtos** do sistema.

### Relatórios
- Geração de relatórios personalizados:
  - **Movimentações:** Produto, Categoria, Quantidade, Tipo de Movimentação e Data.
  - **Estoque:** Produto, Categoria e Quantidade disponível.
  - **Produtos:** Produto, Categoria, Preço e Quantidade.

### Segurança
- **Sistema de login** com autenticação de usuários.
- **Sessões protegidas** para impedir acesso não autorizado.

### Interface
- Tabelas dinâmicas com títulos personalizados de acordo com o relatório selecionado.

---

## Pré-requisitos

Certifique-se de que você possui os seguintes itens instalados:
- **PHP** (versão 7.4 ou superior)
- **MySQL**
- **XAMPP** (ou outro servidor local)
- **Navegador web**
